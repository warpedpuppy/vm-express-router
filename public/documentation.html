My Flix Project

This is an web application that will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Description of the request URL Endpoint HTTP Method Type Parameters in URL Format of data in the body of request Format of the response data
Return a list of all movies to the user /movies GET None None A JSON object holding data about all movies
Return data about a single movie to the user /movies/:Title GET Request parameter - Title of the movie None A JSON object that holds data - genre, description, director's name, and an image URL of the movie requested
Return data about a genre /movies/genre/:Name GET Request parameter - Name of genre None A JSON object holding data about the requested genre - name and description
Return data about a director /movies/director/:Name GET Request parameter- Name of director None A JSON object holding data about the director - name, bio, year of birth, year of death(if not empty)
Allow new users to register /users POST None A JSON object hodling data about the new user - username, password, emailID, and date of birth. Ex: { "Username": "Miriam Myers", "Password": "cVVjmZ72", "EmailId": "mirMyam_mir@gmail.com", "BirthDay": "1991-09-09" } A JSON object holding data about the new user - an userID, username, password, emailId, list of favorite movies(must be empty). Ex: { FavoriteMovies: [], "_id": "5f8fa291fcb1d515507db3e9", "Username": "Miriam Myers", "Password": "cVVjmZ72", "EmailId": "mirMyam_mir@gmail.com", "BirthDay": "1991-09-09", "__v": 0 }
Allow users to update their user info(username) /users/:Username PUT Request parameter - Name of the user whose information needs to be updated. Ex: Thedor Schwann A JSON object holding the updated data - username, password, emailID, and birthday. Ex: { "Username": "Theodora Crain", "Password": "Hi!l3N74a", "EmailId": "theo_crain@gmail.com", "BirthDay": "1982-08-09" } A JSON object holding data - userID, username, password, emailID, and date of birth which reflects the changes made. Ex: { FavoriteMovies: [ "5f8e5de993ad097ba758deab" ], "_id": "5f8e752793ad097ba758dead", "Username": "Theodora Crain", "Password": "Hi!l3N74a", "EmailId": "theo_crain@gmail.com", "BirthDay": "1982-08-09" }
Allow users to add a movie to their list of favorites /users/:Username/favoritemovies/:MovieID POST Request parameter- Movie ID of the movie to be added. None A JSON object of the user reflecting the changes made in the favorite movies list.
Allow users to remove a movie from their list of favorites /users/:Username/favoritemovies/:MovieID DELETE Request parameter- Movie Id of the movie to be removed None A JSON object of the user reflecting the changes made in the favorite movies list.
Allow existing users to deregister /users/:Username DELETE Request parameter- Username of the user None A text message indicating that the user has been deregistered
